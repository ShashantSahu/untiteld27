Minor-2 Project: ASCII Art Representation of James Hetfield

1. Project Overview and Objectives
Late one evening, a student guitarist sat in front of a computer, Metallica riffs echoing softly in the room. Inspired by James Hetfield, the powerful frontman of Metallica, the student wondered:

“Can programming be used as a form of art?”

This project answers that question.
Overview
This project focuses on creating a visual representation of James Hetfield using ASCII characters, generated and displayed through a simple computer program.
Objectives


To understand how text characters can form images


To apply basic programming concepts


To blend music inspiration with computer science


To demonstrate creativity using ASCII code



2. Basic Theory and Prerequisites
Before starting the journey, the student needed a few tools.
ASCII Theory


ASCII (American Standard Code for Information Interchange) assigns numeric values to characters like:


Letters (A–Z, a–z)


Numbers (0–9)


Symbols (@ # $ % *)




When arranged carefully, these characters can create images, called ASCII art


Prerequisites


Basic understanding of:


Text editors


Programming syntax


Print/output statements




Any beginner-friendly language (Python is used here)



3. Motivation and Real-World Relevance
